Git

  • git init - Инициализирует пустой репозиторий Git в текущем каталоге. После этого все изменения будут отслеживаться системой контроля версий.
  • git clone - Клонирует удаленный репозиторий на локальный компьютер.
  • git add - Добавляет файл в индекс (staging area), готовя его к коммиту.
  • git commit -m"" - Добавляет файл в индекс (staging area), готовя его к коммиту.
  • git status - Показывает текущее состояние рабочего дерева, включая незакоммиченные изменения и файлы, которые были добавлены в индекс.
  • git branch - Список всех веток в репозитории. Если добавить параметр , то будет создана новая ветвь с указанным именем.
  • git pull - Загружает последние изменения из удаленного репозитория и объединяет их с текущей ветвью.
  • git push - Отправляет локальные коммиты в удаленный репозиторий. Обычно требуется после завершения работы над изменениями.
  • git remote add origin - Привязывает локальный репозиторий к удаленному.
