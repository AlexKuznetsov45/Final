Git

  • git clone ссылка с GitHub - клонируем удаленный репозиторий с GitHub на локальнй компьютер.
  • git add - добавление файла в отслеживаемые.
  • git status - проверка статуса файла.
  • git commit -m "first commit" - осуществление коммита (операция записи изменений в репозиторий Git), где "first commit" - сообщение для коммита.
  • git log - выведение списка всех коммитов.
