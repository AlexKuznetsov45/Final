Командная строка Windows

  • dir - перечисляет все файлы в текущей директории.
  • cd - позволяет сменить текущую директорию.
  • echo %cd% - показывает текущую рабочую директорию.
  • md - создает новую директорию.
  • move - перемещает или переименовывает файлы.
