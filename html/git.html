Git

  • git status - позволяет просмотреть какие изменения произошли в выбранной директории.
  • git add - позволяет добавить выбранные файлы в отслеживаемые.
  • git commit - позволяет показать что файл изменен для git. Коммит - операция записи изменений в репозиторий git.
  • git push - вносит изменения на удаленный репозиторий.
  • git log - выводит список всех наших коммитов.
